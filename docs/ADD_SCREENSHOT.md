# Screenshots for the README

The README references two images in this folder:

- `ui_globe.png` — ⬜ **please add this one.**

To create `ui_globe.png`:

1. Run the web app: `python web/web_app.py`
2. Open http://localhost:8600
3. Take a screenshot of the page (header + globe + scenario cards) and save it
   here as `docs/ui_globe.png`.

That's the only manual asset the README needs.
